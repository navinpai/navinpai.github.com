Rule Breaking as a Game Design Practice
  • Rule breaking is itself a type of game design schema
  • There are many ways to play a game, and many ways to break/bend the rules
  • NEVER take player behaviour for granted
  • Let game players think like game developers and vice versa
  • Players should not just play within the game, but with the game itself
